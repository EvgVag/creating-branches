Общая информация:

Тип: установка
Языки: мульти, есть русский
Лечение: не требуется

Параметры командной строки:

Тихая установка: /S
Выбор места установки: /D=ПУТЬ

Ключ /D=ПУТЬ следует указывать самым последним
Например: файл_установки.exe /S /D=C:\MyProgram
